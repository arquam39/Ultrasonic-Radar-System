# 📡 Ultrasonic Radar System

The **Ultrasonic Radar System** is a project that detects objects using ultrasonic sensors and displays their distance and position in a radar-style visualization.

This project demonstrates how sensors can be used to detect obstacles and measure distances.

--- 

## 🎯 Project Objectives

* Detect objects using ultrasonic sensors
* Measure distance accurately
* Visualize object detection similar to a radar system
* Understand sensor-based systems

---

## ⚙️ Features

* Real-time object detection
* Distance measurement
* Radar-style scanning
* Sensor-based obstacle detection

---

## 🧰 Components Used

* Arduino Uno
* Ultrasonic Sensor (HC-SR04)
* Servo Motor
* Breadboard
* Jumper Wires
* Power Supply
